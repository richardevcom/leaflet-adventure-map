## About

This project demonstrates clean TypeScript architecture with modular managers for filtering, search, maps, and UI. Built with performance in mind - debounced interactions, optimized builds (~471 KB total, ~102 KB gzip).

Perfect for travel platforms, adventure tourism sites, or any interactive location browser.
